# OKE Command Playbook

## Audit Existing Before Creating

```powershell
py -m OKE source <keyword>
py -m OKE trace <keyword>
dir OKE\<folder>
```

## OKE Health / Release

```powershell
py -m OKE release
```

## Export Continuity

The release command generates, exports, validates, and opens the final package.
