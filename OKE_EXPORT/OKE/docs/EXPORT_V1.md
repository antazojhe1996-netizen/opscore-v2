# Purpose

The Export System is responsible for producing a complete engineering review package for OPSCORE.

Its purpose is not only to export source code, but also to preserve engineering knowledge, architecture, documentation, and AI context.

---

# Philosophy

Export should make a new engineer or a new AI session understand the project without guessing.

Every export must preserve:

- Engineering knowledge
- Architecture
- Documentation
- Context
- Source code
- Review artifacts
